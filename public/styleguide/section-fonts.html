3.3 Relative fontsizes

Use these classes to quicky change the fontsize

  • .t1 - largest font
  • .t2 - second largest font
  • .t4 - fourth largest font
  • .s1 - first font smaller than normal
  • .s2 - second font smaller than normal

Example

This is a font size

This is a font size

 <p class="t1">This is a font size</p>
 <p class="t2">This is a font size</p>
 <p class="t4">This is a font size</p>
 <p class="s1">This is a font size</p>
 <p class="s2">This is a font size</p>
</div>
